# SPA Vanilla

Single Page Application con Vanilla JavaScript

### Instalación 🔧

npm install

## Ejecutar el ambiente de desarrollo ⚙️

npx webpack-dev-server

## Despliegue 📦

npx webpack -d

## Construido con 🛠️

* [Node.JS](https://nodejs.org/es/)
* [WebPack](https://webpack.js.org/)
* [Bootstrap](https://getbootstrap.com/)
